Set Card Game:

- This project is an implantation  of Set card game as described here : https://en.wikipedia.org/wiki/Set_(card_game)
- The game supports both human and AI play (AI is a simple button generator)
- This project was developed as part of a university course at Ben Gurion University - SPL course, and the emphasis was on concurrent programming in Java
- All features in config file are supported - besides featureSize and featureCount
- This assignment got a grade of 100

Enjoy!
